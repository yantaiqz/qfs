### 快速部署 (Quick Start)

#### 1\. 克隆仓库

```bash
git clone [您的仓库链接]
cd [您的仓库名]
```

#### 2\. 安装依赖

```bash
# 建议先创建并激活虚拟环境
pip install streamlit google-genai
```

#### 3\. 配置 API Key

应用需要访问 Gemini API。请在项目根目录创建 `.streamlit/secrets.toml` 文件，并填入您的密钥：

```toml
# .streamlit/secrets.toml
GEMINI_API_KEY="YOUR_API_KEY_HERE"
```

#### 4\. 运行应用

```bash
streamlit run app.py
```
